# イノベーション推進チーム 月次サマリー作成

Slack・Notion・Googleからイノベーション推進チームの月次振り返りを作成する。

## 引数
$ARGUMENTS にはオプションで対象月を指定できる（例: "2月", "2026年3月"）。
指定がない場合は前月を対象とする。

## チーム構成
- 相馬さん（DIDIM事業メイン・展示会・代理店・実証実験・メディア対応）
- 外山さん（DIDIM契約・法務・海外・M&A）
- 鈴村（プロダクト営業・インフラ整備・展示会準備）

## 重要ルール
- **相馬さん・外山さん・鈴村の3名のいずれかが関わっていない項目は含めない**
- 教育プログラム案件（JR東海、サントリー、N高等）は教育チームの取り組み→除外
- 経営企画部の取り組み（宮田さん等）は除外
- 情報収集時は相馬さんのDIDIM関連活動を特に注意して拾う（サイト公開、PR、メディア掲載、代理店契約、実証実験等）

## 手順

### 1. Notion から情報収集
MCP 経由で以下を検索:
- 経営企画部定例の議事録（対象月の全週分）→外山さんの発表欄を確認
- イノベーション推進関連の会議メモ
- DIDIM関連のページ更新

### 2. Slack から情報収集（Notion AI Search経由）
以下のチャンネルを対象に検索:
- #innovation_general
- #innovation_random
- #all_sales（イノベーション推進関連の投稿のみ）

### 3. Google カレンダー・Gmail確認
```bash
python3 ~/check_calendar.py
python3 ~/check_gmail.py
```
対象月の会議・メールからイノベーション推進関連を抽出

### 4. サマリー作成
以下のテンプレートで作成（簡潔に）:

```
## イノベーション推進チーム [対象月]サマリー

### 概要
- 箇条書きで3〜5行程度。主要トピックを簡潔に。

### 成果
- 具体的な成果を箇条書き
- メディア掲載・PR・サイト公開等の外部発信も含める
- 代理店契約・商談の進捗も含める
- 担当者名は省略可（チーム全体の成果として記載）

### 課題
- 課題名
    - →具体的な次アクションを併記

### [翌月]のアクション
- 日付入りで具体的に記載（例: 3/26 幼稚園訪問（埼玉））
- URL等の参考情報があれば添付

### 相談事項
- チームとして相談したい事項
```

### 5. ユーザーに確認
- 作成した草案を提示し、追加・修正を確認
- 特に相馬さんのDIDIM関連で拾えていない情報がないか確認

## 注意事項
- 形式は簡潔に（他チームのサマリーに合わせる）
- 課題には「→」で次アクションを必ず併記
- 翌月アクションは日付入りで具体的に
- PR TIMES、X、メディア掲載等の外部発信URLも成果に含める
- 不確実な情報は含めず、ソースが確認できたもののみ記載
